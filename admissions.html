Introduction to Operating Systems

This course introduces students to the basics of operating systems, covering essential concepts like process management, memory allocation, and the role of the OS in handling user and system interactions.

Basic Computer Skills

This foundational course provides essential skills in using a computer, including navigating the operating system, file management, and understanding hardware basics.

Introduction to Word Processing

Students learn to create and format documents using word processing software, covering editing techniques, formatting styles, and document organization for professional use.

Introduction to Spreadsheets

This course covers the basics of spreadsheet software, focusing on data entry, basic formulas, sorting, and creating simple charts for data visualization.

Introduction to Presentations

Learn how to create engaging presentations using slides, images, animations, and text formatting. This course focuses on creating visuals that support effective communication.

Internet and Email Basics

This introductory course covers safe internet browsing, effective search techniques, and email management, focusing on both personal and professional email etiquette.
